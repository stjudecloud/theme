Designer & Developer resource for building St. Jude Cloud Apps/Portals

Ready to start building apps and portals within St. Jude Cloud?

The gudelines below outline the brand standards for building St. Jude Cloud applications, including global elements (such as header and footer), colors, fonts, etc.

Goals

We aim to create a consistent and easy-to-use framework for designers and developers with a mindset of a responsive and device-compatible front-end design.

In support of the Information Services Strategic Plan, our goals are:

  • Seek ways to increase efficiences while reducing technology complexities
  • Continue our mobile-first approach to application deployment
  • Expand our process-improvement effort

Branding & Global Elements Overview

Global Top Bar - The "Omnibar"

The "Omnibar" - top navigation area - is a consistent unifying element which must be present across all St. Jude Cloud apps and contains St. Jude Cloud logo, global icons and access to St. Jude Cloud applications. Refer to the top header section in this demo site as a visual guide.

  • The St. Jude Cloud logo is composed of a logo symbol (child graphic) followed by a vertical separator and a logotype. This configuration must be used together as a core identity for St. Jude Cloud products and should not be altered. Check with the design team for any questions, specifications or if you need image assets for your application.
  • St. Jude Cloud global icons - TBA
  • St. Jude Cloud apps dropdown menu - TBA

Site-Specific Navigation

Secondary navigation (located right below the top global bar) is designed to house site-specific navigation.

Global Footer

The global footer must appear on all St. Jude Cloud applications and contains copyright information, global links and social media icons. Refer to the footer information in this demo as a visual guide.

Fonts

Typography Primary

To protect our brand and underscore our distinctive approach to everything we do, we have created our own customized brand font, St. Jude Sans. With its choice of font weights, St. Jude Sans can be used for everything from bold headlines and large-scale messaging to elegant body copy.

Typography – Alternative
Use Arial when the brand font is not available.

Icons

We currently use Font Awesome Icons as our base icon set.

Colors

The "St. Jude Cloud Blue" in the top navigation area is the most recognizable branding element for St. Jude Cloud applications. In addition, the colors below are officially-approved St. Jude brand colors. For more detailed information and print color versions, see the St. Jude Branding Guide.
