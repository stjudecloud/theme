Omnibar

The Omnibar is St. Jude Clouds primary navigation. It is used on every app or site, always at the top of each view.

Sketch File

To use the omnibar in your sketch files, you can go to the design.stjude.cloud sketch.cloud directory and choose which screen size template you would like to start your prototype or design.
